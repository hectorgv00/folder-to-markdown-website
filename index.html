ZIP to markdown

Insert a zip of the folder of your project and get a markdown file

Choose a zip file or drag it here

This tool will convert the folder structure of your project into a markdown file!

The markdown file will contain the folder structure of your project in a tree format, and all the comments and docstrings will be included in the markdown file as documentation.

This tool is useful for documenting your project structure. You can use the markdown file to share the project structure with your team members or to create documentation for your project.
